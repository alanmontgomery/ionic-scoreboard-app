# ionic-scoreboard-app
An example of a scoreboard app built using Ionic and React

If you'd like to support, you can <a className="link" href="https://www.buymeacoffee.com/ionicreacthub" target="_blank" rel="noopener">buy me a coffee</a> ☕️

![Ionic Scoreboard App UI](https://repository-images.githubusercontent.com/390244002/f2ae1f3c-3661-465a-9e60-c17f4cfc576c)

### Included in this Ionic React Template/UI
* Add scoreboards & players
* Track simple scores
* Confetti effect on finish
* Pullstate & Ionic CSS utilities

### To run

```javascript
npm install
ionic serve
```

Alternatively, you can add the iOS, Android platform and run natively.
